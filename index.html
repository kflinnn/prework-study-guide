HTML

  • The head element contains information about the webpage. The head stores the metadata.
  • The body element represents the visible content shown to the user
  • Metadata is data that the browser uses to effectively serve the webpage to users. It's not seen by users, but contains information about accessibility, search engines, and performance.

CSS

  • CSS, Cascading Style Sheets, is a programming language used by developers to define how webpage content is presented to users.
  • Three types of CSS are inline, internal style sheet, and external style sheet.
  • Inline styles apply to a specific HTML tag, good for quick and permanent changes, but less flexible than the other two
  • Most common way to style a webpage is using an external CSS style sheet because it allows develpers to keep all their CSS rules in a separate file, making design changes easier.
  • Separation of concerns refers to separatinig code into different files based on use. Based on the idea that each section of code should have its own responsibility.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • "git" needs to be entered before every Git command.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: swithces to indicated branch
  • git add -A: commits all files in the working branch.
  • git commit -m: associates a message with the commit. Add the message in quotes after the -m.

JavaScript

  • A variable is named container that allows us to store data in our code.
  • To declare a variable use the var keyword followed by a name we give the variable. Needs to be unique.
  • Strings include all the items falling into the category. i.e. topics would be HTML, CSS, Git, JavaScript
  • All strings must be surrounded by quotation marks and each line of JavaScript ends in a semicolon.
  • Control flow is the order in which a computer executes code in a script. JavaScript is read from top to bottom, and order counts. Be intentional about setup!
  • Can use conditional statements and loops to control the order the code is executed.
  • An array is a single variable that is used to hold a group of data, typically data that is related in some way. Arrays are zero-indexed and sequential
