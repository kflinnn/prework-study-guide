HTML

  • The head element contains information about the webpage. The head stores the metadata.
  • The body element represents the visible content shown to the user
  • Metadata is data that the browser uses to effectively serve the webpage to users. It's not seen by users, but contains information about accessibility, search engines, and performance.

Git

  • "git" needs to be entered before every Git command.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: swithces to indicated branch
  • git add -A: commits all files in the working branch.
  • git commit -m: associates a message with the commit. Add the message in quotes after the -m.
